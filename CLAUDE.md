# Sloppify POS App

Community-built app powered by Sloppify. Features are proposed and voted on by the community, then implemented by AI agents.

## Tech Stack
- Next.js 16 (App Router)
- Tailwind CSS 4
- TypeScript
- pnpm

## Rules for AI Agent

- Use pnpm, not npm
- Keep changes minimal and focused on the task
- Run `pnpm build` to verify changes compile before committing
- Do NOT make external API calls or fetch requests to third-party domains
- Do NOT install packages that make network requests to external services
- Do NOT add analytics, tracking, or telemetry
- All functionality must be client-side only (no backend/API routes unless explicitly requested)
- Keep the app simple and lightweight
